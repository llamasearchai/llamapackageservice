# Installation

## Using pip

The recommended way to install LlamaPackageservice is via pip:

```bash
pip install llamapackageservice
```

## From Source

You can also install LlamaPackageservice from source:

```bash
git clone https://github.com/llamasearchai/llamapackageservice.git
cd llamapackageservice
pip install -e .
```

## Dependencies

LlamaPackageservice requires:

- Python 3.8+
- Other dependencies are automatically installed by pip
